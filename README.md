# 퀴즈 게임

Vite + React + TypeScript로 만든 퀴즈 게임 웹 애플리케이션입니다.

## 기능

- **퀴즈 풀기**: 퀴즈를 풀어보세요
- **아이템 구매**: 아이템을 구매하세요
- **아이템 강화**: 보기를 선택해서 아이템을 강화하세요
- **전투**: 전투를 시작하세요

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 미리보기

```bash
npm run preview
```
